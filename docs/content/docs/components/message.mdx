## Layout

Message has **variant** (user / ai), **size** (default / sm / lg), and **state** (default / loading / error). Class names are left empty so you can style via your own CSS. The root element gets `data-variant`, `data-size`, and `data-state` for attribute selectors.

```tsx
import { Message } from "@/components/ui/message";

export function Example() {
  return (
    <div className="flex flex-col gap-2">
      <Message variant="user">User message</Message>
      <Message variant="ai" state="loading">AI thinkingâ€¦</Message>
      <Message variant="ai" size="sm" state="error">Error</Message>
    </div>
  );
}
```

With `asChild`, the Message merges its props and styles onto the child element:

```tsx
<Message asChild>
  <article>Rendered as article with message styles</article>
</Message>
```
