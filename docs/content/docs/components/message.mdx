## Layout

Message is a container for chat-style messages. Use `variant` (user/ai), `align` (start/center/end), and `streaming` for AI responses. Style via `className` or `data-slot` selectors.

```tsx
import { Message, MessageContent } from "@/components/ui/message";

export function Example() {
  return (
    <div className="flex flex-col gap-2">
      <Message variant="user" align="end">
        <MessageContent>User message</MessageContent>
      </Message>
      <Message variant="ai" align="start">
        <MessageContent>AI response</MessageContent>
      </Message>
    </div>
  );
}
```

Use `MessageActions` and `MessageAction` for action buttons (e.g. copy, retry). Set `MessageActions variant="hover"` for hover-reveal. `MessageAction` renders as a button with `variant`, `size`, and `asChild` support:

```tsx
import { CopyIcon } from "lucide-react";
import { Message, MessageContent, MessageActions, MessageAction } from "@/components/ui/message";

<Message variant="user" align="end">
  <MessageContent>User message</MessageContent>
  <MessageActions variant="hover">
    <MessageAction variant="ghost" size="icon-sm">
      <CopyIcon className="size-3.5" />
    </MessageAction>
  </MessageActions>
</Message>
```

## Streaming

Set `streaming={true}` for AI responses that are being generated. This adds a shimmer effect and fades the trailing text edge.

